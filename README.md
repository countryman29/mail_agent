# mail_agent

Python-утилита для работы с почтой через IMAP/SMTP.

## Что умеет
- читать письма из папок IMAP
- анализировать ветки переписки
- формировать задачи по письмам
- подготавливать черновики ответов
- отправлять письма через SMTP
- сохранять отправленные письма в папки Sent / Отправленные, если они доступны
- экспортировать вложения

## Основные файлы
- `mail_read_folder.py` — чтение писем из папки
- `mail_threads.py` — группировка по веткам
- `mail_analyze_threads.py` — анализ веток
- `mail_analyze_tasks.py` — формирование задач
- `mail_prepare_reply.py` — подготовка черновика ответа
- `mail_send_reply.py` — отправка письма и сохранение в Sent
- `mail_export_attachments.py` — экспорт вложений
- `mail_folders.py` — работа со списком почтовых папок
